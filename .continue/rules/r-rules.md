# R Rules

## Figure Plotting

* Use ggplot2 for all figure plotting tasks.
* Ensure all ggplot2 figures have a title, axis labels, and a legend.
* Use theme_set() to set a consistent theme for all figures.

## Data Processing

* Use tidyr for data tidying and transformation tasks.
* Use dplyr for data manipulation and summarization tasks.
* Ensure all data processing tasks are performed in a reproducible manner.

## Table Creation

* Use gtsummary or gt for all table creation tasks.

## Code Style

* Use consistent naming conventions for variables, functions, and packages.
* Ensure all code is properly commented and documented.
* Use the style guide for R provided by the tidyverse.
